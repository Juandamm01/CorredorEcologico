# Proyecto Web: Corredor Ecológico - Villavicencio

Este proyecto tiene como objetivo principal concienciar y promover sugerencias para la mejora y preservación del corredor ecológico en Villavicencio, combinando funcionalidades interactivas con una base de datos segura y una interfaz intuitiva.

## Funcionalidades Principales

- Sistema de Registro e Inicio de Sesión
  - Registro de usuarios con contraseñas hasheadas para mayor seguridad.
  - Inicio de sesión y cierre de sesión.
  - Posibilidad de editar la contraseña o eliminar la cuenta desde el perfil.

- Módulo de Sugerencias
  - Solo usuarios registrados e iniciados pueden realizar sugerencias.
  - Las sugerencias pueden ser calificadas de 1 a 5 estrellas por otros usuarios.
  - Posibilidad de ordenar las sugerencias por:
    - Las más recientes.
    - Mejor calificación.

- Integración con Google Maps
  - Muestra la ubicación del corredor ecológico para una mejor contextualización.

- Aviso Legal
  - Sección incluida antes del inicio de sesión (por ahora con texto de ejemplo).

## Tecnologías Utilizadas

- HTML
- CSS
- JavaScript
- Node.js (con Express)
- MySQL
- Git (control de versiones)
